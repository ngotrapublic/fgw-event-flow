# UI/UX Design Workflow

This workflow invokes the **UI/UX Designer Agent** to create beautiful and usable interfaces.

## When to Use

- When designing new user interfaces
- When improving user experience
- When creating design systems
- When reviewing UI implementations

## Instructions

1. **Read the designer agent instructions**:
   - Read the file `.gemini/agents/ui-ux-designer.md` for detailed guidance

2. **Understand requirements**:
   - Identify user needs and goals
   - Analyze existing design patterns
   - Consider accessibility requirements

3. **Design the solution**:
   - Create wireframes or mockups
   - Define color schemes and typography
   - Design responsive layouts
   - Consider micro-interactions

4. **Apply design principles**:
   - Visual hierarchy
   - Consistency
   - Affordance
   - Feedback

5. **Document the design**:
   - Create design specifications
   - Define component styles
   - Document interaction patterns

## Design Principles

- **User-centered**: Design for the user, not for yourself
- **Accessibility**: Design for all abilities (WCAG)
- **Consistency**: Use consistent patterns throughout
- **Simplicity**: Remove unnecessary complexity

## Modern Design Trends

- Glassmorphism
- Dark mode
- Micro-animations
- Gradient accents
- Card-based layouts

## Output Format

Provide design documentation with:
- Design mockups or descriptions
- Color palette and typography
- Component specifications
- Interaction patterns
- Accessibility considerations
