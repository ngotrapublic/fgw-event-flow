# MCP Management Workflow

This workflow invokes the **MCP Manager Agent** to handle MCP server operations.

## When to Use

- When setting up MCP servers
- When configuring MCP connections
- When troubleshooting MCP issues
- When adding new MCP capabilities

## Instructions

1. **Read the mcp manager agent instructions**:
   - Read the file `.gemini/agents/mcp-manager.md` for detailed guidance

2. **Assess MCP requirements**:
   - Identify needed MCP capabilities
   - Check existing configurations
   - Plan new server setup

3. **Configure MCP servers**:
   - Set up server configurations
   - Configure connection parameters
   - Test connectivity

4. **Validate setup**:
   - Test MCP tools functionality
   - Verify resource access
   - Check error handling

5. **Document configuration**:
   - List available servers
   - Document capabilities
   - Note configuration details

## MCP Configuration

MCP servers are configured in `.gemini/settings.json` or project-specific config files.

## Common MCP Tasks

- Setting up filesystem access
- Configuring external API integrations
- Managing MCP tool permissions
- Troubleshooting connection issues

## Output Format

Provide MCP report with:
- Servers configured
- Capabilities available
- Connection status
- Issues resolved
