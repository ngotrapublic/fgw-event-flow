# Brainstorming Workflow

This workflow invokes the **Brainstormer Agent** to generate creative ideas.

## When to Use

- When starting a new project
- When exploring solution alternatives
- When stuck on a problem
- When seeking innovation

## Instructions

1. **Read the brainstormer agent instructions**:
   - Read the file `.gemini/agents/brainstormer.md` for detailed guidance

2. **Define the challenge**:
   - Clearly state the problem or opportunity
   - Identify constraints and requirements
   - Set scope for ideation

3. **Generate ideas**:
   - Use divergent thinking
   - Quantity over quality initially
   - Build on previous ideas
   - No judgment during generation

4. **Evaluate ideas**:
   - Apply convergent thinking
   - Score against criteria
   - Identify most promising options

5. **Refine and present**:
   - Develop top ideas further
   - Create actionable recommendations
   - Present alternatives with pros/cons

## Brainstorming Techniques

- **Mind mapping**: Visual exploration
- **SCAMPER**: Substitute, Combine, Adapt, Modify, Put to other uses, Eliminate, Reverse
- **Six Thinking Hats**: Different perspectives
- **Reverse brainstorming**: What would make it worse?

## Output Format

Provide brainstorming report with:
- Problem statement
- Ideas generated
- Top recommendations
- Next steps
