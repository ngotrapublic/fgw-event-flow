# External Scout Workflow

This workflow invokes the **Scout External Agent** to explore external resources.

## When to Use

- When researching external APIs
- When exploring third-party documentation
- When evaluating external services
- When integrating external systems

## Instructions

1. **Read the scout external agent instructions**:
   - Read the file `.gemini/agents/scout-external.md` for detailed guidance

2. **Identify external resources**:
   - List APIs and services to explore
   - Gather documentation URLs
   - Note authentication requirements

3. **Explore documentation**:
   - Read API documentation
   - Identify available endpoints
   - Note rate limits and restrictions

4. **Evaluate integration**:
   - Assess complexity of integration
   - Identify potential issues
   - Estimate implementation effort

5. **Report findings**:
   - Summarize capabilities
   - Document integration requirements
   - Provide recommendations

## External Resource Checklist

- [ ] Official documentation
- [ ] API reference
- [ ] Authentication methods
- [ ] Rate limits
- [ ] Pricing/usage tiers
- [ ] SDK availability
- [ ] Example implementations

## Output Format

Provide external scout report with:
- Resources explored
- API capabilities summary
- Integration requirements
- Recommendations
- Reference links
