# Copywriting Workflow

This workflow invokes the **Copywriter Agent** to create compelling content.

## When to Use

- When writing marketing copy
- When creating user-facing text
- When writing documentation
- When crafting communications

## Instructions

1. **Read the copywriter agent instructions**:
   - Read the file `.gemini/agents/copywriter.md` for detailed guidance

2. **Understand the brief**:
   - Identify target audience
   - Understand tone and voice
   - Define key messages

3. **Research and plan**:
   - Study existing content
   - Gather relevant information
   - Create content outline

4. **Write the content**:
   - Use clear, engaging language
   - Follow brand guidelines
   - Include calls to action
   - Optimize for readability

5. **Review and refine**:
   - Check for clarity
   - Verify accuracy
   - Proofread for errors

## Writing Principles

- **Know your audience**: Write for them
- **Be clear**: Simple words, short sentences
- **Be specific**: Concrete over abstract
- **Be scannable**: Headers, bullets, whitespace

## Content Types

| Type | Purpose | Tone |
|------|---------|------|
| UI Copy | Interface text | Concise |
| Docs | Technical guides | Clear |
| Marketing | Promotion | Engaging |
| Email | Communication | Professional |

## Output Format

Provide content with:
- Final copy
- Variations (if requested)
- Usage guidelines
