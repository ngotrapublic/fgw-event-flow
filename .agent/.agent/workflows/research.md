# Research Workflow

This workflow invokes the **Researcher Agent** to investigate and gather information.

## When to Use

- When learning about new technologies
- When comparing solutions
- When investigating best practices
- When gathering requirements

## Instructions

1. **Read the researcher agent instructions**:
   - Read the file `.gemini/agents/researcher.md` for detailed guidance

2. **Define research scope**:
   - Clarify the research question
   - Identify key areas to investigate
   - Set boundaries for research

3. **Gather information**:
   - Search documentation
   - Review existing implementations
   - Check best practices

4. **Analyze findings**:
   - Compare options
   - Identify pros and cons
   - Evaluate trade-offs

5. **Report conclusions**:
   - Summarize key findings
   - Provide recommendations
   - List sources and references

## Research Methods

- **Documentation review**: Official docs, guides
- **Code analysis**: Existing implementations
- **Web search**: Articles, tutorials, discussions
- **Comparison**: Feature matrices, benchmarks

## Output Format

Provide research report with:
- Research question
- Key findings
- Comparison table (if applicable)
- Recommendations
- Sources
